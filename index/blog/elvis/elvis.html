Les différents Formats d'Images

Nous allons voir les différents formats d'images les plus courants, leur caractéristique et le domaine d'utilisation suivant nos besoins.

Formats WEB :

- JPG ou JPEG (Joint Photographic Expert Group) : JPG est un format courant avec une compression de données, de taille légère, au format idéal pour la publication sur le Web ou pour l'envoi de mail.
Les inconvénients => À chaque fois que le JPG est de nouveau compressé, l'image se retrouve avec une perte de qualité. De plus, le JPG ne supporte pas le fond transparent.

- PNG (Portable Network Graphics) : Le PNG est un format volumineux, compressé mais sans pertes qui supporte des images avec un fond transparent, de haute qualité, en conservant la couleur et la netteté de l'image initiale. La W3C recommande l'utilisation des formats PNG pour les logos, les icônes, et les images avec du texte.
Les inconvénients => Le PNG pèse plus lourd que JPG, il est donc déconseillé de charger des images dans ce formats sauf pour des utilisations bien précise comme le préconise la W3C.

- GIF (Graphics Interchange Format) : C'est un format très utilisé sur le Web grâce à sa fonction la plus populaire qui est l'animation. Approprié pour stocker des graphismes avec peu de couleurs comme des diagrammes, des logos, ou des formes simples. Si votre logo contient trois couleurs, ou moins, le format GIF convient parfaitement car cela vous permettra de l'utiliser sur toutes les plateformes de par sa petite taille. Et tout comme le PNG, le GIF supporte parfaitement les fonds transparents.
Les inconvénients => Par exemple, une image d'un coucher de soleil sera non seulement trop lourde, mais en plus les couleurs seront déformées à cause de la limitation.

Traitements d'images

- SVG (Scalable Vector Graphics) : Les images vectorielles SVG sont légères et permettent un redimensionnement sans pertes lors de la création d'un site web responsive. Il est également utilisé pour des reproductions graphiques.
Les inconvénients => Il est possible de rencontrer un problème de taille d'un navigateur à un autres.

- PSD (Document PhotoShop) : Principalement utilisé avec le logiciel pour retoucher des images pixelisées, non vectorielles. L'image est de qualité sans perte.
Les inconvénients => Il est déconseillé pour une utilisation Web car c'est un fichier lourd.

- TIFF (Tagged Image File Format) : Format non compressé, sans aucune perte de qualités qui supporte des fonds transparents. Les images aux formats TIFF sont utilisées principalement dans le domaine professionnel, comme l'imprimerie et la publicité.
Les inconvénients => Très lourd, il ne convient pas pour une utilisation Web.

- AI (Adobe Illustrator Document) : Utilisé par les graphistes et les imprimeurs pour des créations visuelles, c'est un format sans pertes de qualité et sans pixellisation.
Les inconvénients => Il n'est compatible qu'avec le logiciel Adobe illustrator.

- PDF (Portable Document Format) : Généralement utilisé pour de la lecture de document et des formulaires. On peut facilement les stocker et les partager. C'est un fichier qui peut contenir du texte, des vidéos, des images, des animations … On peut ainsi intégrer des images vectorielles et garder leurs qualités même en compressant le fichier PDF.
Les inconvénients => Une fois l'image enregistrée dans ce format, il est difficile de modifier les données stockées.
